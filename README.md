# Indian Employee Data Cleaning 🧹📊

## 📌 Overview
This project focuses on cleaning and preprocessing a self-collected **Indian Employee Dataset** using **Python (Pandas & NumPy)**. The goal was to transform raw, inconsistent data into a clean, structured, and analysis-ready dataset.

## 📂 Dataset
- The dataset was **self-collected/created**, containing employee-related information (e.g., names, departments, salary, experience, etc.).
- Since it was raw data, it required significant cleaning before it could be used for analysis or modeling.

## 🛠️ Tools & Technologies
- **Python**
- **Pandas** – for data manipulation and cleaning
- **NumPy** – for numerical operations
- **VS Code** – as the development environment

## 🔧 Data Cleaning Steps
The following cleaning operations were performed on the dataset:

1. **Handling Missing Values**  
   Identified and treated missing/null values using appropriate techniques (e.g., imputation or removal).

2. **Removing Duplicate Rows**  
   Detected and removed duplicate entries to ensure data integrity.

3. **Fixing Column Names & Formatting**  
   Standardized column names (removed extra spaces, fixed casing) and corrected inconsistent formatting in the data.

4. **Converting Data Types**  
   Ensured each column had the correct data type (e.g., converting strings to numeric/date formats where needed).

5. **Handling Outliers**  
   Identified outliers in numerical columns and treated them to avoid skewed analysis.
